相關步驟如下:<br>
1. 結合維基百科、新聞、百度等文章進行簡體轉繁體後，進行斷詞與詞向量模型訓練<br>
2. 建立LSTM神經網路
3. 將網路上的店家評論放入神經網路之輸入層，經運算德出該評論為正向、中立或是負向情緒<br>
(因訓練文本、詞向量模型過於巨大無法上傳)
